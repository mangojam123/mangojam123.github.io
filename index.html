ROCIS Outdoor Data Explorer

Built a public-facing web application for the ROCIS Low-Cost Monitoring Project that visualizes outdoor particle data from Dylos sensors. Implemented Python routines to clean and aggregate raw logs, align timestamps, calculate duration and missing-data ratios, and generate cohort-specific CSV files used by the site. Developed the front end in HTML, CSS, JavaScript, and Plotly.js with client-side CSV loading, dataset-type and cohort selection, interactive time-series plots, and log/linear scale toggles. I continue to update both the data pipeline and interface to keep the tool reliable and user-friendly for ROCIS staff and community participants.

Speech-Based Emotion Recognition

Developed a speech emotion recognition system using MFCC, Chroma-STFT, and spectral features, trained on a 1,440-file emotional speech corpus. Implemented preprocessing (noise removal, stretching, pitch/shift augmentation) and feature-extraction pipelines in Python and trained an MLP classifier. Achieved 73% classification accuracy across seven emotion classes and designed an end-to-end workflow for parsing, preprocessing, feature engineering, and evaluation.

Bias and Fairness Analysis in Early Alzheimer's Detection

Analyzed the OASIS dataset using Logistic Regression, Random Forest, XGBoost, LightGBM, and CatBoost to detect bias in Alzheimer's diagnosis models. Evaluated performance using balanced accuracy and ROC-AUC, and conducted subgroup fairness analysis across gender, age, SES, education, and intersectional groups. Applied Fairlearn mitigation methods and used SHAP feature attributions to verify that clinical variables such as MMSE, education, age, and nWBV primarily drove predictions rather than demographic attributes.

Predictive Modeling for Paint Popularity (PPG Industries Collaboration)

Built regression, decision-tree, and clustering models to analyze RGB/HSL features from PPG's historical color dataset and forecast paint-popularity trends. Engineered feature sets from 30k+ color records, performed preprocessing and exploratory analysis, and evaluated models with metrics such as RMSE and silhouette score. Generated insights on hue, saturation, and brightness patterns that supported data-driven product and marketing decisions.
